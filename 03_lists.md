# LISTS

<!-- UNORDER LIST -->
## Unorder lists
List without specific order.  
We use `*` symbol for each item.

### Example
* Item 1
* Item 2
* Item 3

**Syntax:**  
`*` Item 1  
`*` Item 2  
`*` Item 3


<!-- ORDER LIST -->
## Order lists
List with specific order.  
We use `NUMBER.` symbol for each item.

### Example
1. Item 1
2. Item 2
3. Item 3

**Syntax:**  
`1.` Item 1  
`2.` Item 2  
`3.` Item 3
