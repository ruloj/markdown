# HYPERLINKS

Hyperlinks are used to link our file to another file or website.  
To create hyperlinks, we use `[text](link 'title')`.

### Example  
Go to [Google](https://www.google.com.mx "Ir a google")

**Syntax:**  
Go to `[Google](https://www.google.com.mx "Ir a google")`
